参数生成规则#第一步获取接口密钥从管理控制台获取 APIKey 和 APISecret，均为 32 位字符串。#第二步：构造 signature 原始字段signature_origin 由 host、date、request-line 三个参数按以下格式拼接（\n 为换行符，: 后有一个空格）：host: $host\ndate: $date\n$request-line以创建任务接口为例：请求 https://api.example.com/api/v1/video/generate// apiKey / apiSecret: 控制台获取func assembleAuthUrl(hosturl, apiKey, apiSecret string) string {    ul, err := url.Parse(hosturl)    if err != nil {        fmt.Println(err)    }    // 签名时间（UTC RFC1123）    date := time.Now().UTC().Format(time.RFC1123)    // 构造签名原始字段    signString := []string{        "host: " + ul.Host,        "date: " + date,        "POST " + ul.Path + " HTTP/1.1",    }    sgin := strings.Join(signString, "\n")    // hmac-sha256 签名并 Base64    sha := HmacWithShaToBase64("hmac-sha256", sgin, apiSecret)    // 拼接 authorization_origin    authOrigin := fmt.Sprintf(        `api_key="%s", algorithm="hmac-sha256", headers="host date request-line", signature="%s"`,        apiKey, sha,    )    authorization := base64.StdEncoding.EncodeToString([]byte(authOrigin))    v := url.Values{}    v.Add("host", ul.Host)    v.Add("date", date)    v.Add("authorization", authorization)    return hosturl + "?" + v.Encode()}#鉴权失败响应HTTP Code	说明	错误描述	处理建议401	缺少 authorization 参数	{"message":"Unauthorized"}	检查是否正确携带 authorization 参数401	签名参数解析失败	{"message":"HMAC signature cannot be verified"}	检查各签名参数是否完整，确认 APIKey
